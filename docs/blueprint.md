# **App Name**: 校园智联

## Core Features:

- 学生档案与偏好: 允许新生和老生创建详细的学生档案，包括学术背景、兴趣、技能和偏好，以便进行有效的匹配。
- 智能同伴匹配: 利用智能匹配工具，根据共同兴趣、学术需求和职业抱负，将新生与高年级导师连接起来，提供个性化指导。
- 资源共享中心: 提供一个集中的平台，供老生分享课程笔记、学习指南和职业建议等宝贵资源，新生可以轻松发现和访问相关内容。
- 个性化成长路径: 提供一个分析工具，根据新生的档案和早期参与数据，提供量身定制的建议、任务推荐和指导，帮助他们适应大学生活和学习。
- 互动反馈与进度跟踪: 实施一个系统，供学生跟踪他们在导师项目中的进展，接收实时反馈，并根据参与数据调整他们的学习和适应策略。
- 数字成长顾问: 开发一个基于数据模拟的顾问功能，根据历史数据模拟学生的潜在成长路径和结果，并提供预测性洞察。

## Style Guidelines:

- 主色调：温馨智慧的蓝色 (#3D9FDB)，用于关键交互元素，象征信任与学术专注。
- 背景色：柔和、高度去饱和的浅蓝色 (#E8F5FC)，营造干净、开放、亲切的界面，适合学习平台使用。
- 强调色：充满活力、互补的青蓝色 (#21A99F)，少量用于高亮、行动召唤按钮，增添现代动态感。
- 标题和正文字体：'Inter' (无衬线字体)，因其现代、简洁和高度可读性而被选用，适用于网络和微信小程序界面的多样化内容。
- 采用一套现代、简约的图标，清晰表达连接、成长、指导和数字互动等概念。强调清晰度和直观性。
- 采用简洁的卡片式布局来组织学生档案、资源和任务，确保在不同设备（网页和微信小程序）上的响应性。
- 为导航、内容加载和用户反馈融入微妙流畅的过渡动画，提升整体用户体验，同时避免分散注意力。
